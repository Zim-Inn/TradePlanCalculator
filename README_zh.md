# 股票交易计划计算器

这是一个基于 Web 的客户端工具，旨在帮助股票投资者根据其现有持仓和市场预期，快速制定和评估潜在的交易计划。它通过动态计算不同情景下的买入/卖出价格点，为日内或短期交易决策提供数据支持。

## ✨ 功能特性

- **动态交易计划生成**: 根据您设置的期望盈利、支撑位等参数，自动计算实现目标所需的买入或卖出价格。
- **多种数据加载方式**:
    - **文件导入**: 支持通过 Tab 分隔的 `.csv` 文件  快速导入您的全部持仓。
    - **手动管理**: 可在界面中直接手动添加、编辑和删除单条持仓记录。
- **灵活的计算参数**:
    - **基准交易量**: 可自由设定用于生成交易计划的基准股数（单位：手），而非始终基于总持仓。
    - **涨跌停幅**: 自定义次日价格波动的限制，用于风险评估。
    - **预计支撑位/期望总盈利**: 核心计算参数，用以反推交易价格。
- **交互式持仓列表**:
    - 支持对加载的持仓数据进行**全选**、**反选**和**删除**操作。
    - 支持**在线编辑**持仓数量、成本价和当前价，计算结果会实时联动更新。
- **实时风险校验**:
    - 在您输入参数时，系统会实时判断输入的价格是否超出次日涨跌停范围，并给出醒目提示。
    - 在最终生成的计划中，也会高亮标记出在涨跌停限制下的可执行与不可执行的操作。
- **纯客户端，无后端依赖**: 零外部依赖，所有计算均在您的浏览器中完成，确保了数据隐私和安全性。

## 🚀 如何使用

1.  **打开应用**
    - 下载本项目，直接在您喜欢的现代 web 浏览器中打开 `股票交易计划计算器.html` 文件即可开始使用。

2.  **第一步: 加载并选择持仓**
    - **方式一：文件加载**:
        - 准备一个符合格式要求的文本文件, 一般而言, 你可以通过券商的电脑版交易软件导出持仓, 选择txt或csv格式导出即可.Excel也有将表格处理为CSV的功能
        - 文件内容需使用 **Tab (制表符)** 分隔，并包含以下必需的表头和数据列: `证券代码`、`证券名称`、`证券数量`、`成本价`、`当前价`。
        - 点击 **"点击选择 '当前持仓.csv' 文件"** 按钮并选择您的文件。
    - **方式二：手动添加**:
        - 点击 **"手动添加持仓"** 按钮。
        - 在展开的表单中填写股票信息，然后点击 **"确认添加"**。
    - **数据交互**:
        - 加载后的数据列表是可交互的。您可以直接点击修改**持仓数量**、**成本价**和**当前价**，或点击**删除**按钮移除某条记录。
    - **选择持仓**:
        - 勾选您希望为其制定交易计划的一支或多支股票。
        - 点击 **"下一步"** 按钮。

3.  **第二步: 填写参数并计算**
    - 为您选择的每支股票填写独立的计算参数。
        - **预计支撑位** 默认填充为该股票的当前价。
    - 所有参数填写完毕后，点击 **"开始计算"**。

4.  **第三步: 查看综合方案**
    - 系统将为您选择的每支股票生成一个包含**减仓计划 (卖出)**和**加仓计划 (买入)**的综合方案卡片。
    - 方案会清晰地列出在不同交易股数下，为达到您的期望盈利，需要设定的具体挂单价格。
    - 同时，系统会根据您设定的涨跌停幅，用醒目的颜色标记出哪些计划在次日是理论上可执行的。
